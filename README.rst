trace_link
==========

What?
-----

Trace a (spam) link through a redirection chain.

How?
----

* Python's subprocess module
* lxml.html Python package
* curl

Supported
---------

* Response Header's ``Location``
* ``http-equiv``
* ``refresh``
* manual

Planned
